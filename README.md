# Akash
DoseGuardian is an embedded system designed to help patients take medicines on time.
It uses a microcontroller with RTC to track time and provide alerts using LCD, buzzer, and LED.
This system is especially useful for elderly people and patients who often forget their medication.

To remind users to take medicines on time
To reduce missed doses
To provide a simple and reliable healthcare solution
⚙️ Components Used
LPC2148 Microcontroller
16x2 LCD
RTC Module
4x4 Keypad
Buzzer
LED
Switches

🔧 Working
User sets medicine time using keypad
RTC keeps track of current time
When time matches:
LCD displays “Take Medicine Now”
Buzzer gives alert
User presses switch to acknowledge
If not pressed → LED indicates missed dose
✨ Features
Real-time medicine reminder
Easy to use
Accurate timing using RTC
Missed dose indication
🚀 Applications
Home healthcare
Hospitals
Elderly care

✅ Conclusion

DoseGuardian helps improve medication adherence and reduces health risks by providing timely alerts.
